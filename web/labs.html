Lab 2: Responsive Design and Single Page Design

For this lab I learned how to design a website as a virtul single page. This means that in essence I have three distinct pages in my website. However, using some javascript and css I am able to present all these 'pages' in one scrollable webpage.

I also had the opportunity to learn about responsive design. Responsive design means that a website alters its' layout according to the size of the viewing device. So the same website will look different on a laptop and a mobile phone, for example. I achieved this by using media queries and the jQuery javascript framework.

Lab 3 & 4: Custom JavaScript Animation Framework

In this lab I created a custom Javascript framework. The framework allows a programmer to create and animate cat images. The framework uses sprites and allows the user to animate cats walking around on the screen.

I learned a lot about JavaScript in this lab. I learned a lot of JS syntax, how JS objects relate to and differ from objects in other languages. I also learned about frameworks, what they are and how to make my own.
