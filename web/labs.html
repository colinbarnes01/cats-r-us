Lab 2: Responsive Design and Single Page Design

For this lab I learned how to design a website as a virtul single page. This means that in essence I have three distinct pages in my website. However, using some javascript and css I am able to present all these 'pages' in one scrollable webpage.

I also had the opportunity to learn about responsive design. Responsive design means that a website alters its' layout according to the size of the viewing device. So the same website will look different on a laptop and a mobile phone, for example. I achieved this by using media queries and the jQuery javascript framework.

Lab 7: CSS Keyframe Custom Framework Tutorial

In this lab I created my own custom CSS Keyframe framework. I used JavaScript to enhance the simple functionality of CSS Keyframes to allow users to create custome CSS Keyframe animations using JavaScript. Users of the framework are able to pass parameters to their animations via HTML input, as well as restart their animations and play them repeatedly.

I learned a lot about CSS and how animation can be done using Keyframes. I also learned how JavaScript can access and alter the DOM of a webpage, and how to create style elements in the head element using JavaScript.
